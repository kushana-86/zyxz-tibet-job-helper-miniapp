# 择域先知

`择域先知` 是我基于“非西藏生源定向西藏就业”这一使用场景开发的一个辅助选岗微信小程序，用来帮助用户更高效地筛选、比较和收藏意向地区与岗位信息。

## 项目简介

这个小程序围绕“选岗参考”来设计，重点提供以下能力：

- 地区与岗位信息检索
- 多地区对比查看
- 收藏候选项，便于反复筛选
- 基于本地数据的快速浏览与分析

## 页面结构

- `pages/home/home`：首页，总览与入口
- `pages/search/search`：搜索页，按条件筛选地区与相关信息
- `pages/compare/compare`：对比页，支持多地区横向比较
- `pages/favorites/favorites`：收藏页，集中管理备选内容

## 数据说明

- 项目当前包含本地数据文件，便于直接运行和调试
- 如需接入云开发数据库，可参考 `database/README.md`
- 数据文件中包含岗位明细与地区辅助信息

## 本地运行

1. 使用微信开发者工具导入本项目
2. 选择项目根目录
3. 按需使用现有 `AppID` 或测试号
4. 编译后即可在模拟器中查看各页面功能
